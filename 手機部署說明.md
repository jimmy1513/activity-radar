# 手機部署說明

這個工具不串 API，也沒有 API key。部署後只是把表單網頁放到一個網址，讓手機可以打開。

## 推薦路線：GitHub Pages

1. 到 GitHub 建立一個新的 public repository。
2. 上傳這些檔案：
   - `index.html`
   - `activity-radar.html`
   - `manifest.webmanifest`
   - `activity-radar-icon.svg`
   - `sw.js`
   - `.nojekyll`
3. 到 repository 的 Settings -> Pages。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，資料夾選 `/root`。
6. 儲存後等待 GitHub 產生網址。

GitHub Pages 免費方案通常需要 public repository；如果要 private repository 發佈 Pages，可能需要付費方案。

## 手機加入主畫面

### iPhone Safari

1. 打開 GitHub Pages 網址。
2. 點分享按鈕。
3. 選「加入主畫面」。

### Android Chrome

1. 打開 GitHub Pages 網址。
2. 點右上角選單。
3. 選「新增至主畫面」或「安裝應用程式」。
